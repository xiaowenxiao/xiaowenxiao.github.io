k8s暴露外部应用服务

外部的应用（ip+端口）以service的方式在k8s集群暴露

>> cat jumpserver.yaml
apiVersion: v1
kind: Service
metadata:
  name: jumpserver-service
spec:
  ports:
    - port: 80
      protocol: TCP
      targetPort: 80

---
apiVersion: v1
kind: Endpoints
metadata:
  name: jumpserver-service
subsets:
  - addresses:
      - ip: "10.10.10.10"
    ports:
      - port: 80
  • 查看
